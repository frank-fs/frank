💡 Key Principles

  • Hypermedia First: Send HTML from the server (Examples 1-5, 7-9)
  • Server as Source of Truth: Let the server decide what to show
  • Minimal Signals: Only for form inputs and ephemeral state (Examples 6, 10)
  • Avoid Client State: Don't store application data in signals
