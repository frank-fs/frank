# Contracts: OPTIONS and Link Header Discovery

This feature is an F# library extension, not a service with external API endpoints.

The "contracts" are the F# type signatures and middleware behaviors documented in:
- `data-model.md` -- type definitions (`DiscoveryMediaType`, middleware constructors)
- `quickstart.md` -- usage patterns and expected HTTP response formats
- `spec.md` -- functional requirements and acceptance scenarios

No OpenAPI, GraphQL, or other external API contract files are applicable.
